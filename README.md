# Carrinho de Produtos - JavaScript

Projeto desenvolvido para praticar conceitos de **JavaScript moderno**, manipulação de **DOM** e **lógica de programação**.

A aplicação simula um pequeno carrinho de compras onde é possível adicionar produtos com nome e preço e calcular automaticamente o valor total.

---

## Funcionalidades

- Adicionar produtos ao carrinho
- Validação de campos
- Validação de preço
- Impedir produtos duplicados
- Cálculo automático do valor total
- Exibição dinâmica na tela

---

## Conceitos praticados

Este projeto foi desenvolvido para praticar:

- Classes em JavaScript
- Getters e Setters
- Métodos estáticos
- Manipulação do DOM
- Eventos (`submit`)
- Estrutura de dados `Map`
- Organização de código em funções

---

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript (Vanilla JS)

---

## 🎯 Objetivo do projeto

O objetivo deste projeto foi praticar a integração entre **HTML, CSS e JavaScript**, além de melhorar habilidades de **lógica de programação e organização de código**.
